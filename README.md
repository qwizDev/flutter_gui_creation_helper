# Package: gui_creation_helper

A Flutter package that shall provide widgets and helper classes for making flutter gui-development faster and easier.
So it aims at providing things that are needed or useful for almost every flutter app.
Thus, it aims at providing reusable components, a standardized "design-theme", and more.
